# Antigravity性能テスト用

これは、Google Analyticsの性能を測るため、作らせたゲームです。

## 使用プロンプト
```
web-gameディレクトリに対して、下記の優先度で可能な限り高度なゲームを作ってほしいです。
1.フロントエンドはWebブラウザで表示できる
2.ゲーム性を持ち、可能な限り面白く
3.初見でも分かりやすいUI/UX
4.可能な限り処理は軽量に
5.複数人（2，3人）で同時に遊べる（サーバーが必要であれば、専用で組み、php,jsかVPSでのホスティングとする）
ゲーム自体のプランニングから、行ってください
```

## ローカルネットワーク配信
同一LAN内の他端末からアクセスできるよう、サーバーはデフォルトで `0.0.0.0` にバインドします。

### 起動
```bash
npm install
npm start
```

### アクセス
- 同じPC: `http://localhost:3000`
- 同じLAN内の他端末: `http://<サーバーPCのローカルIP>:3000`

### ホスト/ポートの変更
```bash
HOST=0.0.0.0 PORT=3000 npm start
```
